Chapter 1. Ext4FS module overview

This is the Ext4 FS module documentaion of the Spruce system. This module is designated to cover as much ext4fs-specific code as possible. One of the most important segments of such code is the ioctl system call backend. In case of Ext4 FS it is called ext4_ioctl and is situated in fs/ext4/ioctl.c.

The module covers the following Ext4fs-specific features:

  • Getting and setting flags
  • Clearing non-allowed flags
  • Getting and setting version
  • Online resize
  • Online defragmentation
  • Delayed block allocation
  • Test on unsupported operation
