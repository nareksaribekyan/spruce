Chapter 1. Overview

Spruce is a Linux file system checker. It consists of the following modules:

  • Main module
  • Syscall checker module
  • Common operations module
  • Benchmark module
  • Main module
  • and a set of file system specific modules:
    • Ext4 FS module
    • BTRFS module
